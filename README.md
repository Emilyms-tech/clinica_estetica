# Site de Clínica Estética

Este projeto é um site institucional simples de uma clínica estética.  
Ele apresenta algumas informações sobre os procedimentos oferecidos pela clínica e também disponibiliza uma página de contato.

🔗 **Acesse o site:**  
https://emilyms-tech.github.io/clinica_estetica/

## Funcionalidades

- Página inicial com apresentação da clínica
- Informações sobre procedimentos estéticos
- Página de contato com email, telefone e endereço
- Navegação entre as páginas do site

## Tecnologias utilizadas

- HTML5  
- CSS3

## Estrutura do projeto

clinica_estetica/

- index.html  
- contato.html  
- CSS/  
- imagem/

## Objetivo do projeto

O objetivo deste projeto foi praticar a criação e organização de páginas web e a estilização utilizando CSS.
